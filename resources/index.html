This section contains miscellaneous Stata resources, including some legacy software.

bundle. An old program to bundle images on a web page using base64 encoding. conditionals. A follow up to a Stata Forum post on simulating multivariate normal variates. micombine. A note on multiple imputation discussing how to combine the estimates from first principles. optim. Mata's `optimize` made simple, with an example estimating frequency of blood types. rules. Some JavaScript to prettify your Stata posts on the web using IBM drawing characters. weave. Old code to weave Stata output and annotations, superseded by `markstat`. whereis. The `whereis` command, which also has a readme in GitHub.
