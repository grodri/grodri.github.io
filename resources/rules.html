It's easy as 1-2-3:

1. Make sure your output blocks are preformatted with class "stata".

2. Include the following CSS rule in your page or style file:

pre.stata {
	font-family "Lucida Console", monospace; font-size:13px; line-height:13px;
}

3. Include the following script at the bottom of your page: 

<script src="https://grodri.github.io/assets/js/strules.js"></script>

Remarks

The script works by detecting horizontal and vertical rules (and their crossings) and rewriting them using the original IBM drawing characters: 

─   │   ┌   ┬   ┐   ├   ┼   ┤   └   ┴   ┘

Of course you need a monospace font that has these Unicode characters. On Windows I get the best results with "Lucida Console", which is why this is the first choice. Otherwise, looks like the best strategy is simply to specify "monospace" and trust the browser to make a good choice. This seems to work on Windows 10, Mac OS X, Linux Ubuntu 16.4 and even my iPhone 5. Feedback is always appreciated.
